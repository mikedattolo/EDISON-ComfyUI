📚 Setup Guide — Engineering Laptop

Connect your Windows 11 laptop (i7 / RTX A3000) running Rhino 7, SolidWorks, and Blender to EDISON as a CAD worker node.

  1. Install Python 3.10+ from python.org — check “Add Python to PATH”
  2. Copy the agent — copy the tools/node-agent/ folder from this repo to your laptop
  3. Run the installer — double-click install_windows.bat, or open a cmd window: 
    cd C:\path\to\node-agent
install_windows.bat
  4. Or run manually (replace with your EDISON server IP): 
    cd C:\path\to\node-agent
pip install -r requirements.txt
python edison_node_agent.py --server 192.168.1.X --name "CAD-Laptop" --role cad
  5. Rhino 7: open Rhino before starting the agent so it can connect via COM
  6. SolidWorks: open SolidWorks before starting the agent; also install: 
    pip install pywin32
  7. Blender is auto-detected from PATH — no extra setup needed
  8. Auto-start (optional): run setup_autostart.ps1 in PowerShell

The agent auto-detects CPU, RAM, GPU, and installed software, registers with EDISON, and sends heartbeats every 30 s.
