📖 Quick Setup Guide

Connect your Windows engineering laptop to EDISON

  1. Install Python 3.10+ on the laptop if not already installed — python.org
  2. Download the agent files — copy the tools/node-agent/ folder from your EDISON server to the laptop
  3. Install dependencies: 
    cd node-agent
pip install -r requirements.txt
  4. Run the agent (replace with your EDISON server's IP): 
    python edison_node_agent.py --server YOUR_EDISON_IP --name "Engineering-Laptop" --role cad
  5. For Rhino 7 integration, also install pywin32: 
    pip install pywin32
    Then start Rhino 7 before running the agent.
  6. Auto-start (optional): Create a shortcut in your Startup folder or use Task Scheduler to run the agent at login.

The agent will auto-detect your CPU, RAM, GPU, and installed software (Rhino, Blender, etc.) and register with EDISON. It sends heartbeats every 30 seconds to stay connected.
